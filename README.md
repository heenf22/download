# Download List

## [주식 농부(Stock Farmer)][download-stockfarmer]

주식 농부 프로그램은 저평가된 주식을 찾을 수 있도록 정보를 제공합니다. 현재는 대한민국 주식 및 ETF 정보만 제공합니다.

* Support OS : Windows
* Dependancy : [.NET 6.0][download-dotnet-6]
